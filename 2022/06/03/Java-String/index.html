д№ӢжүҖд»ҘеҮәзҺ°иҝҷж ·зҡ„з»“жһңпјҢжҳҜеӯ—з¬ҰдёІзҡ„еӯҳеӮЁж–№ејҸдёҚеҗҢгҖӮеӯ—з¬ҰдёІstr1е’Ңstr11иҝҷж ·зҡ„еҲӣе»әж–№ејҸжҳҜе°Ҷж•°жҚ®еӯҳж”ҫеңЁеӯ—з¬ҰдёІеёёйҮҸжұ пјҢеңЁеёёйҮҸжұ дёӯпјҢеҗҢдёҖдёӘж•°еҸӘжңүдёҖд»ҪгҖӮеӯ—з¬ҰдёІеёёйҮҸжұ еңЁJVMдёӯжҳҜStringTableзұ»пјҢе®һйҷ…жҳҜдёҖдёӘеӣәе®ҡеӨ§е°Ҹзҡ„HashTableгҖӮstr2е’Ңstr22жҳҜе°Ҷж•°жҚ®еӯҳж”ҫеңЁе ҶдёҠпјҢжҜҸnewдёҖж¬Ўе°ұдјҡеңЁе ҶдёҠејҖиҫҹдёҖеқ—з©әй—ҙгҖӮ
